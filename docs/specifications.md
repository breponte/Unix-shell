# Project Specifications

This Unix shell is capable of running programs, redirecting input/output, and creating pipelines. More advanced features include job control and signal handling.

The general timeline is as follows:
- Program execution via `fork`/`exec`/`wait`
- I/O redirection via `>`, `<`, `>>` via `dup2`
- Pipelined execution
- Signal handling
- Job control
